# 🌐 Los Domínguez Bar • Restaurante Website

This is a custom website I designed and built for a local café-bar in Almería, Spain — **Los Domínguez Bar • Restaurante**.

The goal was simple: give the business a strong online presence that reflects its warm, authentic atmosphere and makes it easy for customers to explore the menu, find the location, and get in touch.

## 🚀 Live Website

👉 http://losdominguez.xo.je

## ✨ Features

* Modern, responsive design (mobile-first)
* Clean and elegant UI inspired by café aesthetics
* Menu showcase with real images
* Gallery section highlighting the place & food
* Contact section with location and phone
* Smooth scrolling and user-friendly navigation

## 💡 About This Project

This project is part of my work creating websites for local businesses that don’t yet have an online presence.
I focus on building simple, stylish, and effective websites that help them attract more customers.
